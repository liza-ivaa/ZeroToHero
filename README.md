# Веб-сервіс для обробки статистичних даних

## Опис проєкту
Наш проєкт — це інтерактивний калькулятор для швидкого аналізу числових вибірок.  
Він автоматизує рутинні обчислення з курсу статистики та теорії ймовірностей, допомагаючи студентам різних напрямів перевіряти свої розрахунки та аналізувати дані.

Основний функціонал сайту забезпечує короткий підсумок про вибірку та про спостереження, які були зроблені. Це дозволяє легко систематизувати та представити первинні дані через показники центральної тенденції, що робить їх значно зрозумілішими.

## Чому проєкт корисний
Сервіс допомагає швидко отримати короткий статистичний підсумок вибірки.  
Користувач може легко систематизувати та представити первинні дані за допомогою основних статистичних показників.  
Це значно спрощує аналіз даних та робить результати більш зрозумілими.

## Можливості системи
Користувачі можуть виконувати такі дії:

- Обчислення основних характеристик:
  - мода
  - медіана
  - середнє значення
- Розрахунок показників варіації:
  - дисперсія
  - середньоквадратичне відхилення
- Побудова візуальних графіків та гістограм на основі введених даних
- Формування висновку на основі аналізу отриманих даних
